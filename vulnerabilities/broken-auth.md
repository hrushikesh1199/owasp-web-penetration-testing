# Broken Authentication

## Description
Broken authentication vulnerabilities occur when authentication and session
management mechanisms are improperly implemented.

---

## Observations
- Weak credential handling in the lab environment
- Predictable authentication behavior

---

## Impact
- Account takeover
- Unauthorized access
- Privilege escalation

---

## Severity
High

---

## Mitigation
- Enforce strong password policies
- Implement secure session management
- Use multi-factor authentication where applicable
