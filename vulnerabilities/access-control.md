# Broken Access Control

## Description
Broken access control occurs when users are able to access resources or functions
outside of their intended authorization level.

---

## Proof of Concept
Access to restricted endpoints without proper authorization checks.

---

## Impact
- Unauthorized data access
- Privilege escalation
- Business logic abuse

---

## Severity
High

---

## Mitigation
- Enforce server-side access control checks
- Implement role-based access control
- Deny access by default
